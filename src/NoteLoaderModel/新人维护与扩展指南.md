# 纸条系统新人维护与扩展指南

## 系统概述

纸条系统是一个模块化的星露谷模组组件，用于加载、显示和管理自定义秘密纸条。系统由多个职责明确的模块组成，通过依赖注入和事件系统实现功能。

**核心功能：**
- 加载自定义纸条数据
- 创建纸条物品
- 显示纸条内容
- 执行纸条动作（如给予物品、添加任务）
- 将纸条添加到收集品列表
- 在收集品页面显示纸条

## 添加新纸条

### 1. 创建纸条数据
在 `assets/notes/secret_notes.json` 文件中添加新的纸条数据：

```json
{
  "NoteDatas": [
    {
      "Id": "note_001",
      "Condition": "always",
      "BackgroundImagePath": "",
      "ContentImagePath": "",
      "ContentText": "这是一张新纸条的内容。\n它可以包含多行文本。",
      "Attachments": [
        {
          "Type": "Item",
          "ItemId": "74",
          "Quantity": 5,
          "Chance": 1.0
        }
      ],
      "TriggerActions": [
        {
          "Type": "AddQuest",
          "Parameters": {
            "QuestId": "example_quest"
          }
        }
      ]
    }
  ]
}
```

### 2. 字段说明
- **Id**: 纸条唯一标识符
- **Condition**: 显示条件（目前仅支持"always"）
- **BackgroundImagePath**: 背景图片路径（可选）
- **ContentImagePath**: 内容图片路径（可选）
- **ContentText**: 纸条文本内容
- **Attachments**: 附件列表（如物品奖励）
- **TriggerActions**: 触发动作列表（如添加任务）

### 3. 测试新纸条
1. 构建并运行模组
2. 在游戏中按 M 键获取第一个纸条
3. 右键点击纸条查看内容
4. 检查是否获得附件物品
5. 检查收集品页面是否显示纸条

## 常见问题和解决方案

### 1. 纸条不显示
- **原因**：纸条数据未正确加载或ID冲突
- **解决方案**：
  - 检查 `secret_notes.json` 文件格式是否正确
  - 查看日志文件中的错误信息
  - 检查ID映射文件是否正确生成

### 2. 纸条物品不可用
- **原因**：物品数据未正确注册
- **解决方案**：
  - 检查 `Data/Objects` 资产是否正确修改
  - 确保物品ID格式正确（`MorticianMod_noteId`）

### 3. 收集品页面不显示纸条
- **原因**：Harmony补丁未正确应用或纸条未添加到收集
- **解决方案**：
  - 检查Harmony补丁是否成功应用
  - 确保纸条使用后调用了 `AddNote` 方法
  - 查看日志文件中的错误信息

### 4. 动作不执行
- **原因**：动作配置错误或处理逻辑有问题
- **解决方案**：
  - 检查 `TriggerActions` 配置是否正确
  - 查看日志文件中的错误信息
  - 调试 `TriggerNoteActions` 方法
