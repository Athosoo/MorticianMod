# 秘密纸条展示功能修复计划

## 问题分析

当前实现能够检测到玩家右键使用秘密纸条，但纸条展示功能缺失。主要问题：

1. **反射调用问题**：使用反射创建 SecretNote 实例可能存在问题
2. **纸条数据注册**：确保纸条数据正确注册到游戏的 Data/SecretNotes
3. **显示机制**：需要使用星露谷原生的纸条显示机制

## 实现计划

- **优先级**: P0
- **依赖**: 无
- **描述**:

  - 纸条数据在游戏中可访问
  - IntId 正确分配且唯一
